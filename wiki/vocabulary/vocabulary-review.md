# Vocabulary Review

This page tracks active vocabulary for guided daily practice. It is not a full dictionary.

## Rating rule

The learner does not need to assign exact scores. The daily page asks for `hard / okay / easy`; the LLM updates the level using both the learner's feeling and actual sentence accuracy.

| level | meaning | evidence |
|---|---|---|
| 0 | unknown | The learner does not recognize the word. |
| 1 | recognized | The learner can recognize a rough meaning. |
| 2 | understood in context | The learner understands it in reading/listening but cannot use it reliably. |
| 3 | usable with support | The learner can use it in a sentence frame or with a phrase prompt. |
| 4 | active use | The learner can use it freely and correctly several times. |

## Status rule

| status | meaning |
|---|---|
| weak | Needs near-term review. |
| active | Current learning item. |
| review | Occasional review only. |
| mastered | Temporarily mastered; do not schedule often. |
