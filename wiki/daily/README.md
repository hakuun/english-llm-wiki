# Daily Study Pages

Use this folder for daily guided study pages.

## Workflow

1. The learner says: 今天学习.
2. The LLM reads [[active-learning]], [[log]], [[profile/current-level]], [[profile/weak-points]], [[profile/error-patterns]], and [[vocabulary/vocabulary-review]].
3. The LLM creates `wiki/daily/YYYY-MM-DD.md` from [[daily/_template]].
4. The learner completes guided tasks: choices, blanks, imitation, small output, stuck points, and difficulty.
5. The learner says: 我完成了，帮我批改并更新知识库.
6. The LLM updates the daily page, [[log]], [[vocabulary/vocabulary-review]], and other pages when needed.

## Learner input rule

The learner should not need to invent a blank free-writing topic. Daily pages should provide sentence frames and easy options.
