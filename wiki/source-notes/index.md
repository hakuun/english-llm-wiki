# Source Notes

This folder stores notes extracted from processed source materials.

## Core pages

- [[source-notes/authoritative-materials-map]] — How to find, judge, and use authoritative English learning materials.

## Source note format

Each source note should include:

- source title,
- source type,
- approximate CEFR level,
- why it is useful,
- key vocabulary,
- key grammar or sentence patterns,
- recommended study action.
