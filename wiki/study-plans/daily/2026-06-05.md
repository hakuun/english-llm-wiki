## 3. 先讲清楚：为什么动词要加 -s？

### 一条你昨天问的好问题

你问：「为什么动词会加 s？表示复数吗？」

答案：**恰恰相反！动词加 -s = 单数标记。**

这是英语最反直觉的地方之一，因为名词的规则刚好反过来：

```
名词：book（单数）→ books（复数）    -s = 复数 ✅
动词：I weigh（复数主语）→ He weighs（单数主语）  -s = 单数 ✅
```

**口诀：名词加 -s 变复数，动词加 -s 变单数（只用于 he/she/it）。**

### 什么时候加 -s？

英语动词的 -s 只出现在一种情况：**主语是第三人称单数**。

| 主语类型              | 人称         | 例子                           | 动词形式      |
| ----------------- | ---------- | ---------------------------- | --------- |
| I                 | 第一人称单数     | I weigh 60kg.                | 原形（不加 -s） |
| You               | 第二人称       | You weigh 60kg.              | 原形（不加 -s） |
| We / They         | 复数         | They weigh 60kg.             | 原形（不加 -s） |
| **He / She / It** | **第三人称单数** | **He weighs** 60kg.          | **加 -s**  |
| **单数名词**          | **第三人称单数** | **The bag weighs** 5kg.      | **加 -s**  |
| **My suitcase**   | **第三人称单数** | **My suitcase weighs** 10kg. | **加 -s**  |

### 什么时候绝对不加 -s？

`does / can / must / will / should` 这些**助动词/情态动词**后面，动词永远用**原形**。

```
✅ Does it weigh 5kg?        （does 后面 → weigh，不加 -s）
✅ It does not weigh much.    （does 后面 → weigh）
✅ You must weigh your bag.   （must 后面 → weigh）
✅ Can you weigh this?        （can 后面 → weigh）
```

**记法：does 自己已经带了 -s（do → does），所以后面的动词不用再带。**

类比中文：你已经说了一次「了」，就不会再说第二次。
- Does（已经变了） + weigh（就不用变了）

## 7. Confusable pairs — 昨天学到的两对

昨天输出暴露了两组好词。今天快速巩固一下。

### quiet vs quit

| 词 | 意思 | 例句 |
|---|---|---|
| **quiet** | 安静（形容词） | The library is very **quiet**. |
| **quit** | 退出/戒掉（动词） | He wants to **quit** smoking. |

**记法**：qu**ie**t 里有 **e** → 安静的"**E**"（嗯……很安静）。quit 没有 e → 走了，退出了。

选词填空：

1. Please be __quiet___ in the classroom. (quiet / quit)
2. I want to __quit___ my job. (quiet / quit)
3. You must keep __quiet___ in the library. (quiet / quit)

### clean vs clear

| 词         | 意思           | 例句                                                        |
| --------- | ------------ | --------------------------------------------------------- |
| **clean** | 干净（卫生方面）     | The room is **clean**.                                    |
| **clear** | 清楚/清晰（视觉/理解） | The water is **clear**. / The instructions are **clear**. |

**记法**：cle**a**n → **a** = 阿姨打扫干**净**。cle**a**r → **a**ir = 空**气**很清**澈**。

选词填空：

1. Please keep the classroom __clean___. (干净 → clean / clear)
2. The sky is very __clear___ today. (晴朗/清澈 → clean / clear)
3. Is the meaning __clear___ now? (意思清楚了吗 → clean / clear)

#### 回答你的疑问：为什么 My suitcases 不加 -s？

你问："My suitcases 不也是 it 吗？这里怎么不加 s？"

**关键点：suitcase 是一个物品，但 suitcases 后面有 -s → 它不是 it，是 they！**

```
My suitcase = it（单数）   →  weighs  ← 动词加 -s
My suitcases = they（复数） →  weigh   ← 动词不加 -s
```

名词后面的 **-s** 直接决定了它的人称：

- 名词无 -s（单数名词如 suitcase, bag, the box）→ = it → **动词加 -s**
- 名词有 -s（复数名词如 suitcases, bags, the boxes）→ = they → **动词不加 -s**

所以你的口诀可以扩展成完整的双向规则：

> **名词加了 -s → 主语是复数 → 动词不加 -s**
> **名词没加 -s → 主语是单数(it) → 动词加 -s**

这就是为什么"名词 -s = 复数，动词 -s = 单数"这条规则在句子层面是自洽的——名词和动词的 -s 不重复出现，它们各管各的。
