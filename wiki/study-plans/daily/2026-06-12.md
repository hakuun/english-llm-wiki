## 4. Input — Listening Day 1（12 分钟）

### 听前提示

这篇文章讨论一个有趣的问题：**为什么有些动物是黑白的？**（斑马、熊猫、企鹅、虎鲸……）

核心观点：
- 斑马的黑白条纹 → **驱赶苍蝇**（optical illusion 让苍蝇看不清）
- 熊猫的黑白毛色 → **雪地伪装**（远处看和石头、雪混在一起）
- 臭鼬的黑白花纹 → **警告捕食者**"我有毒，别吃我"
- 科学家最后的结论 → 没有唯一答案，每种动物不一样

## 7. Pattern in context（1 分钟）

### 从听力中学一个实用结构：It turned out that...

听力中科学家说：**It turned out that** every black and white pattern was really effective.

> **It turned out that...** = 结果发现……（用来描述做了一件事之后发现的结果）

例句：
- We tried many methods, but **it turned out that** the simplest one worked best.
- I thought the test was hard, but **it turned out that** it was quite easy.

**Q3b 拼写说明**：`swiming` → `swimming`。规则：短元音 + 单辅音结尾的动词，加 -ing 时要**双写最后一个辅音字母**。

| 原形 | -ing 形式 | 规则 |
|------|-----------|------|
| swim | swi**mm**ing | 短元音 i + 单辅音 m → 双写 m |
| run | ru**nn**ing | 短元音 u + 单辅音 n → 双写 n |
| sit | si**tt**ing | 短元音 i + 单辅音 t → 双写 t |
| get | ge**tt**ing | 同上 |
| help | helping | ❌ 不双写（已有两个辅音 lp） |
| learn | learning | ❌ 不双写（长元音 + 两个辅音 rn） |

> 不需要一次记住，遇到时注意即可。今天先记住最常见的：**swim → swimming, run → running, sit → sitting, get → getting**。
