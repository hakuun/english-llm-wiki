## 3. 先聊一下：为什么 weigh 是动词？

昨天你问了一个非常好的问题：**为什么一个东西有多重，"重"会是动词？**

这不是你的问题——这是中英文的根本差异。我们换个角度想。

### 英语句子 = 必须有动词

一个完整的英语句子，**必须有一个动词**。没有动词就是残废句。

```
这个箱子 ___ 8公斤。
 主语      动词空位    数字
```

这个空位只能填**动词**。就像「我 ___ 饭」的空位只能填「吃」一样——这是动词的专属位置。

不同语言给这个动词空位填的词不一样：

| 语言 | 句子 | 空位填了什么词 |
|---|---|---|
| 中文 | 这个箱子 **重** 8 公斤。 | 「重」——它在这个位置**临时当了动词** |
| 英文 | This suitcase **weighs** 8 kg. | weigh（动词） |

中文的「重」可以在不同位置自由切换身份，不需要变形。但英文不行——动词位置必须用一个真正的动词。

### 试试这个感觉

选出每个句子的**动词**（做动作的那个词）：

| 句子 | 动词是？ |
|---|---|
| 我 **吃** 饭。 | 吃 |
| 他 **喜欢** 猫。 | 喜欢 |
| 这个包 **重** 5 公斤。 | 重 |

看到了吗？中文「这个包重 5 公斤」里，「重」就是在做动词的活。只是因为你从小就说中文，从来没想过它在这个位置是动词。

### 英文的三个「重」

同一个意思，英文必须分三个词：

```
位置需要形容词 → heavy   （This bag is heavy.）
位置需要动词   → weigh    （This bag weighs 5kg.）
位置需要名词   → weight   （The weight is 5kg.）
```

**记法：看空位前面是什么。**

| 空位前面 | 需要 | 举例 |
|---|---|---|
| `is / are / very / too` | 形容词 | is **heavy**, very **heavy** |
| `主语 (the bag)` | 动词 | The bag **weighs** 5kg. |
| `does / do / can / must` | 动词原形 | does it **weigh**? |
| `the / a / my / its` | 名词 | the **weight**, my **weight** |

今天我们就用这个「看左边」的方法来练 weigh/weight。

## 4. Warm-up — weigh vs weight 二选一

先不急着做题。先看左边是什么，再决定用哪个词。

**规则**：
- 左边是 `is / are / very` → 需要**形容词** → heavy
- 左边是 `主语` 或 `does/can/must` → 需要**动词** → weigh / weighs
- 左边是 `the / a / my` → 需要**名词** → weight

**先判断，再填空**。每道题先回答「空位需要什么词性？」，再写答案。

| 句子 | 空位需要什么词性？ | 答案（weigh / weighs / weight / heavy） |
|---|---|---|
| 1. The bag is very _____. | 形容词 | |
| 2. My suitcase _____ 10kg. | 动词 | |
| 3. The _____ of the bag is 10kg. | 名词 | |
| 4. How much does it _____? | _____（does 后面跟 ____） | |
| 5. This box is too _____. I can't carry it. | _____ | |

## 5. Core practice 1 — weigh/weight 三选一 + 新配对

### Part A: weigh / weighs / weight

选词填空，每道题先划出「左边是什么」。

1. This bag _____ too much. I need a smaller one.
   - [ ] weigh
   - [ ] weighs
   - [ ] weight

2. The maximum _____ for check-in luggage is 23kg.
   - [ ] weigh
   - [ ] weighs
   - [ ] weight

3. You must _____ your suitcase before you check in.
   - [ ] weigh
   - [ ] weighs
   - [ ] weight

4. How much does your hand luggage _____?
   - [ ] weigh
   - [ ] weighs
   - [ ] weight

### Part B: 扩展到 high / height 和 long / length

weigh/weight 不是唯一的。英语里很多「形容词/动词 → 名词」对都遵循同样的逻辑：

| 形容词 | 动词 | 名词 | 中文 |
|---|---|---|---|
| heavy | weigh | weight | 重 |
| high | — | height | 高 |
| long | — | length | 长 |

**名词的特征：前面常有 `the`、`my`、`its`，后面常有 `of`。**

选词填空：

1. The Eiffel Tower is very _____. (high / height)
   - 左边是 `very` → 需要 _____

2. The _____ of the Eiffel Tower is 330 meters. (high / height)
   - 左边是 `the` → 需要 _____

3. The Yangtze River is very _____. (long / length)
   - 左边是 `very` → 需要 _____

4. The _____ of the Yangtze River is about 6,300 km. (long / length)
   - 左边是 `the` → 需要 _____

### Part C: 混合挑战

从方框里选词填空（每个词只用一次）：

```
weigh   weight   high   height   long   length
```

1. The _____ of the Great Wall is more than 21,000 km.
2. Can you _____ this for me? I don't know how heavy it is.
3. Mount Everest is very _____ — 8,849 meters.
4. The maximum _____ for this elevator is 500kg.
5. This road is very _____. It goes all the way to the next city.

## 6. Core practice 2 — cannot vs must 两列对比

### 规则（记住这两句话）

```
must   = 必须做（不做的后果是你承担不了）
cannot = 不能做（禁止，不允许）
```

中文「不能」= **cannot**，不是 must。

### Part A: 判断是「必须」还是「不能」

先不看英文，只看中文意思：

| 中文 | 这是…… |
|---|---|
| 你必须出示护照。 | 必须 / 不能 |
| 你不能在教室里吃东西。 | 必须 / 不能 |
| 液体不能超过 100ml。 | 必须 / 不能 |
| 飞机起飞前必须系好安全带。 | 必须 / 不能 |
| 你不能在地铁上吸烟。 | 必须 / 不能 |

### Part B: 选 must 还是 cannot

| 句子 | 答案 |
|---|---|
| 1. You _____ show your passport at the airport.（必须出示） | must / cannot |
| 2. You _____ smoke in the classroom.（不能吸烟） | must / cannot |
| 3. You _____ use your phone during takeoff.（不能用手机） | must / cannot |
| 4. You _____ wear a seatbelt in the car.（必须系安全带） | must / cannot |
| 5. Liquids _____ be more than 100ml in your hand luggage.（不能超过） | must / cannot |
| 6. You _____ speak loudly in the library.（不能大声说话） | must / cannot |
