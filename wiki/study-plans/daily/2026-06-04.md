## 3. 先聊一下：为什么 weigh 是动词？

昨天你问了一个非常好的问题：**为什么一个东西有多重，"重"会是动词？**

这不是你的问题——这是中英文的根本差异。我们换个角度想。

### 英语句子 = 必须有动词

一个完整的英语句子，**必须有一个动词**。没有动词就是残废句。

```
这个箱子 ___ 8公斤。
 主语      动词空位    数字
```

这个空位只能填**动词**。就像「我 ___ 饭」的空位只能填「吃」一样——这是动词的专属位置。

不同语言给这个动词空位填的词不一样：

| 语言 | 句子 | 空位填了什么词 |
|---|---|---|
| 中文 | 这个箱子 **重** 8 公斤。 | 「重」——它在这个位置**临时当了动词** |
| 英文 | This suitcase **weighs** 8 kg. | weigh（动词） |

中文的「重」可以在不同位置自由切换身份，不需要变形。但英文不行——动词位置必须用一个真正的动词。

### 试试这个感觉

选出每个句子的**动词**（做动作的那个词）：

| 句子 | 动词是？ |
|---|---|
| 我 **吃** 饭。 | 吃 |
| 他 **喜欢** 猫。 | 喜欢 |
| 这个包 **重** 5 公斤。 | 重 |

看到了吗？中文「这个包重 5 公斤」里，「重」就是在做动词的活。只是因为你从小就说中文，从来没想过它在这个位置是动词。

### 英文的三个「重」

同一个意思，英文必须分三个词：

```
位置需要形容词 → heavy   （This bag is heavy.）
位置需要动词   → weigh    （This bag weighs 5kg.）
位置需要名词   → weight   （The weight is 5kg.）
```

**记法：看空位前面是什么。**

| 空位前面 | 需要 | 举例 |
|---|---|---|
| `is / are / very / too` | 形容词 | is **heavy**, very **heavy** |
| `主语 (the bag)` | 动词 | The bag **weighs** 5kg. |
| `does / do / can / must` | 动词原形 | does it **weigh**? |
| `the / a / my / its` | 名词 | the **weight**, my **weight** |

今天我们就用这个「看左边」的方法来练 weigh/weight。

### 规则（记住这两句话）

```
must   = 必须做（不做的后果是你承担不了）

```

| long | — ❌ | length |

#### 1. 🔴 第三人称单数主谓一致（新暴露的深层问题）

weigh/weight 区分本身有明显的进步——今天 4 个动词空位你全填了动词（weigh/weighs），没有再填 weight。**「先判词性」的习惯正在形成** 🎉。

但在 weigh/weight 练习中挖出了一个更深层的语法问题：**什么时候用原形 vs 加 -s？**

| 今天的错误 | 根因 |
|---|---|
| My suitcase **weigh** 10kg. → 应为 weighs | 主语三单时漏了 -s |
| How much does it **weighs**? → 应为 weigh | does 后面多加了 -s |
| This bag **weigh** too much. → 应为 weighs | 主语三单时漏了 -s |
| How much does your luggage **weighs**? → 应为 weigh | does 后面多加了 -s |

**两条规则打包记**：

```
I/You/We/They weigh     ← 不加 -s
He/She/It    weighs     ← 加 -s（第三人称单数）

does + weigh            ← does 后面原形（does 已经带了 -s，动词不再加）
can/must/will + weigh   ← 情态动词后面原形
```

⚠️ 这个错误模式和原来的 weigh/weight 混淆是**不同的问题**。原来的混淆在好转，但三单 -s 是新暴露的语法弱项，需要单独练习。
