# Daily Study Pages

Use this folder as the single home for daily guided study pages.

## Workflow

1. The learner says: 今天学习.
2. The LLM reviews [[active-learning]], [[log]], [[profile/current-level]], [[profile/weak-points]], [[profile/error-patterns]], [[vocabulary/vocabulary-review]], and recent daily pages.
3. The LLM creates or updates `wiki/study-plans/daily/YYYY-MM-DD.md` from [[study-plans/daily/_template]].
4. The learner completes guided tasks: review, input, vocabulary, sentence imitation, small output, stuck points, and difficulty.
5. The learner says: 我完成了，帮我批改并更新知识库.
6. The LLM updates the daily page, [[log]], [[vocabulary/vocabulary-review]], and related profile pages when useful.
