#### 5. Vocabulary

**1. put off（B1 ⭐）**

```
你的句子：Lemons is so sour that put me off.
```

| 问题 | 说明 | 修正 |
|------|------|------|
| `Lemons is` | 复数主语 + is ❌ → **Lemons are** | Lemons **are** |
| `that put` | 从句缺主语。谁 put me off? → **they** | that **they** put me off |

> `put off` 的用法：主语（什么东西）+ put(s) + someone + off
> ```
> ✅ Boring games put me off.
> ✅ Lemons are so sour that they put me off.  ← that 后面需要主语！
> ```

✅ 修正：**Lemons are so sour that they put me off.**

---

**2. improve（A1）**

```
你的句子：I want to improve my English by hard learn every day.
```

| 问题 | 说明 | 修正 |
|------|------|------|
| `improve` 的使用 | ✅ 完全正确 | — |
| `by hard learn` | by + gerund ❌ → **by studying hard** | by studying hard |

> 🔑 **by + doing 规则回顾**（上次 06-06 修过，06-15 抽查通过，今天又回来了）：
> ```
> by 后面动词要加 -ing
> 
> ✅ by studying hard
> ✅ by reading every day  
> ✅ by listening to podcasts
> ❌ by hard learn  ← 中文"通过努力学习"直译
> ```

✅ 修正：**I want to improve my English by studying hard every day.**

**3. recommend（A2）**

```
你的句子：I would recommend this movie for you.
```

| 问题 | 说明 | 修正 |
|------|------|------|
| 句子结构 | ✅ 基本正确 | — |
| 介词 | `for you` → `to you` 更自然 | recommend this movie **to** you |

> `recommend something to someone` 是标准搭配。`for you` 不算错，但意思是"替你推荐"（不是"向你推荐"）。
> ```
> ✅ I would recommend this movie to you.    ← 向你推荐
> ✅ I would recommend this book to my friend. ← 向朋友推荐
> ```

✅ 修正：**I would recommend this movie to you.**（或保持 `for you` 也可以接受）

💬 骨架完全正确！两个拼写都是手指滑了（fru→fui, cau→cau 反了），你应该都认识这些词。

**句 2** `I don't like lemons, because this sour put me off.`

| 问题 | 你的答案 | 修正 |
|------|----------|------|
| 句子骨架 | ✅ I don't like lemons 正确 | — |
| `this sour` | sour 是形容词，不能当名词用 | the sour **taste** |
| `put` 主谓一致 | this taste = 单数 → **puts** | puts me off |

> 🔑 **形容词 ≠ 名词**（和 weigh/weight 同类的词性问题）：
> ```
> sour = 形容词（酸的），不能说 "this sour"
> 
> ✅ the sour taste   ← taste 是名词，"酸味"
> ✅ they are sour    ← are + 形容词
> ❌ this sour        ← 缺名词
> ```

✅ 修正：**I don't like lemons, because the sour taste puts me off.**
或更简单：**I don't like lemons because they are too sour.**

**句 3** `Mango is more sweet than banner.`

| 问题 | 你的答案 | 修正 |
|------|----------|------|
| 比较句 be 动词 | **is** ✅ 🎉 | — |
| 比较级形式 | `more sweet` → **sweeter** | sweeter |
| 拼写 | `banner` | **banana** |

> 🎉 **最大亮点：比较句没有忘记 be 动词！** 06-22 的 `Three Schools English more expensive than` 缺 is，今天 `Mango is more sweet than` — **is 在！**
>
> 🔑 **比较级规则**（1 分钟小知识点）：
> ```
> 短形容词（1 个音节）→ 加 -er，不用 more
>   sweet → sweeter    big → bigger    hot → hotter
> 
> 长形容词（2+ 个音节）→ 用 more
>   expensive → more expensive
>   delicious → more delicious
> 
> sweet 很短 → sweeter，不用 more sweet
> ```

✅ 修正：**Mango is sweeter than banana.**
