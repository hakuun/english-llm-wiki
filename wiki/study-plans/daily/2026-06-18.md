## 4. Proactive Grammar — have been vs have gone（12 分钟）

> **来源**：Oxford 语法 — Present perfect simple: been and gone（A-level source）
>
> 这是现在完成时第 3 课。直接解决 06-16 你写 `I never gone Hong Kong` 时的 gone/been 混淆。

---

### Part A — 从这两个例子学规则（2 分钟）

先看两组对比：

```
场景 1：
  A: Where's Dad?
  B: He's gone to the supermarket.     ← 爸爸还在超市，没回来

场景 2：
  A: I've been to the supermarket. Do you want an apple?
     ← 我已经从超市回来了，现在人在家
```

再看一组：

```
Andy has been to New York three times.    ← 去过 3 次，都回来了
Anna has gone to Santiago. She'll be back on Friday.  ← 现在在 Santiago，还没回来
```

**问自己**：你能看出 been 和 gone 的区别吗？用一句话写在下面：

```
been = 去了，已经_____________
gone = 去了，还没_____________
```

---

### Part B — 规则总结（用中文看明白）（1 分钟）

| | been | gone |
|------|------|------|
| 意思 | 去过（已经回来了） | 去了（人还在那里） |
| 人现在在哪 | 👈 这里（回来了） | 👉 那里（还没回来） |
| 例句 | Dad's been to the supermarket. = He went and **came back**. | Dad's gone to the supermarket. = He is **still there**. |

> 💡 **关键判断**：说这句话的时候，**人在不在眼前？**
> - 人在眼前 / 已经回来了 → been
> - 人不在眼前 / 还没回来 → gone

**一个额外规则**：问"你有没有去过……？"（ever），**永远用 been**，不用 gone。

```
✅ Have you ever been to Iceland?
❌ Have you ever gone to Iceland?
```

→ 因为问你"去过没"，默认你已经回来了！

> 判断标准：**说这句话的时候，人在哪里？**

```

   人还在银行 → 选：been / gone

2. I have ________ to that restaurant twice. The food is great!
   我去过两次，都回来了 → 选：been / gone

   他刚走，还没回来 → 选：been / gone
```
