### What to repair today

昨天出现了一个新错误，今天是重点修复对象：

1. **情态动词 + 动词原形**（新错误）
   - ❌ `Passengers can only taking one suitcase.`
   - ✅ `Passengers can only take one suitcase.`
   - 规则：`can / must / should / will / cannot` 后面永远用**动词原形**，不加 -ing，不加 to。

2. **`weigh`（动词）vs `weight`（名词）**
   - ❌ The suitcase must **weight** no more than 10kg.
   - ✅ The suitcase must **weigh** no more than 10kg.
   - 记法：`weigh` = 称重（动作），`weight` = 重量（数字）。

3. **`cannot` 和 `must` 不混淆**
   - `must be more than 10kg` = 必须超过 10kg
   - `cannot be more than 10kg` = 不能超过 10kg
   - 限制规则用 `cannot`，不是 `must`。

## 7. Grammar / sentence pattern

### Pattern 1: 情态动词 + 动词原形（⭐ 今日核心）

```
✅ can + base form     → You can take a taxi.
✅ must + base form    → You must pass security.
✅ cannot + base form  → You cannot eat on the train.
✅ will + base form    → You will pay extra.

❌ can + -ing    → can taking ✗
❌ can + to      → can to take ✗
❌ must + -ing   → must passing ✗
```

**记住：情态动词后面永远是原形，不加尾巴。**

### Pattern 2: `weigh` (verb) vs `weight` (noun)

```
weigh (v.)  → The bag weighs 5kg.     （动词：它重5kg）
weight (n.) → The weight of the bag is 5kg. （名词：重量是5kg）

类似：
high (adj.) → The mountain is high.   （形容词：山高）
height (n.) → The height is 3000m.    （名词：高度是3000m）
```
