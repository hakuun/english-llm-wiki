# Daily Routine

Default daily study time: 30 minutes.

## Standard 30-minute session

Use this structure unless the learner gives a different focus for the day.

1. Input: 10-12 minutes
   - Read a short graded text, or
   - Listen to short audio with a transcript.
   - Current default: reading-first, A1-A2 level, gradually moving toward A2-B1.

2. Vocabulary and collocations: 6-8 minutes
   - Learn 6-10 useful words or phrases from the input.
   - Focus on example sentences and common word combinations.
   - Avoid long isolated word lists.

3. Grammar or sentence pattern: 5-7 minutes
   - Study one practical grammar point or sentence pattern from the input.
   - For the current stage, prioritize sentence structure over abstract grammar theory.
   - Make 2-3 simple example sentences.

4. Output: 5 minutes
   - Write 3-5 sentences,
   - or answer 2 simple questions,
   - or summarize/retell the input in simple English.

## Daily completion record

After each session, record these four items:

```text
今日完成：
1. 阅读/听力材料：完成 / 未完成
2. 新词：记住了几个？
3. 输出：写了几句？
4. 最难的地方：单词 / 句子 / 语法 / 听不懂 / 没时间 / 其他
```

When the learner asks for the next daily plan, use this record to adjust difficulty, review load, and output tasks.

## Weekly adjustment trigger

A weekly review should happen when either condition is true:

- The learner asks for a weekly review.
- About 7 daily sessions have accumulated since the last weekly review.

Weekly review should check:

1. Which daily tasks were completed or skipped.
2. Which words, sentence patterns, or grammar points remained difficult.
3. Whether the material level was too easy, suitable, or too hard.
4. Whether the next week should emphasize reading, listening, vocabulary, grammar, or output.

Then update `wiki/study-plans/weekly-plan.md` and append a log entry.

## Monthly assessment trigger

A monthly assessment should happen when either condition is true:

- The learner asks for a monthly assessment.
- About 30 days or 4 weekly reviews have passed since the last monthly assessment.

Monthly assessment should check:

1. Reading level and reading speed.
2. Vocabulary growth and active use of words.
3. Listening comprehension if listening was practiced.
4. Grammar accuracy in learner output.
5. Writing length, clarity, and recurring errors.
6. Current evidence against A2/B1/B2 can-do goals.

Then update `wiki/assessments/monthly-checkpoints.md`, `wiki/profile/current-level.md`, `wiki/study-plans/monthly-plan.md`, and append a log entry.

## If time is only 15 minutes

1. 8 minutes input.
2. 5 minutes vocabulary or sentence pattern.
3. 2 minutes one-sentence output.

## If time is 45-60 minutes

Extend input and output, not only vocabulary review.

## Daily principle

A finished small session is better than an ambitious unfinished session.

The plan should create visible evidence every day: one completed input task, a small vocabulary set, one sentence pattern, and a short output sample.
